# AlertX

## Reducing False Alarms in Real-Time Suspicious Activity Detection

AlertX is an AI-powered surveillance system designed to reduce false alarms in real-time suspicious activity detection. The system uses Computer Vision and Artificial Intelligence techniques to analyze surveillance videos and identify potential threats more accurately.

## Project Objective

The main objective of AlertX is to reduce false alarms generated by traditional surveillance systems and improve security monitoring through intelligent threat detection.

## Project Architecture

1. User uploads surveillance video.
2. Video is processed using YOLOv8.
3. Objects and suspicious activities are detected.
4. Alerts are generated based on threat level.
5. Detection logs are stored.
6. Reports are generated for analysis.
